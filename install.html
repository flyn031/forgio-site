Install Forgio in 5 minutes.

One script tag. Any website. Your AI pre-sales engineer goes live the moment you paste it.

The script tag

Copy this and paste it into your website's header or before the closing </body> tag. We'll customise the values for your business during onboarding.

<!-- Forgio AI Pre-Sales Widget --> <script src="https://forgio.co.uk/widget.js" data-api-url="https://forgio-backend-production.up.railway.app" data-tenant="your-company" data-template="your-industry" data-allow-uploads="true" data-launcher-text="Start your enquiry" data-color="#FF6B35" ></script>
AttributeWhat it does
data-tenantYour unique company ID — we set this up for you
data-templateYour industry template (conveyors, refrigeration, fabrication, etc.)
data-allow-uploadsEnable file and drawing uploads (true/false)
data-launcher-textThe text on the chat button — customise to match your site
data-colorAccent colour — use your brand hex code

Platform guides

Click your platform for step-by-step instructions.

WordPress
2 minutes
  1. Install the free "Insert Headers and Footers" plugin (or "WPCode")
  2. Go to Settings → Insert Headers and Footers
  3. Paste the Forgio script tag into the "Scripts in Header" box
  4. Click Save
  5. Refresh your site — the chat button appears in the bottom-right corner

Alternatively: Appearance → Theme Editor → header.php — paste before </head>

Wix
2 minutes
  1. Go to Settings → Custom Code
  2. Click "Add Custom Code"
  3. Paste the Forgio script tag
  4. Set placement to "All pages" and "Head"
  5. Click Apply
Squarespace
2 minutes
  1. Go to Settings → Advanced → Code Injection
  2. Paste the Forgio script tag into the "Header" box
  3. Click Save
  4. Refresh your site to confirm
Shopify
3 minutes
  1. Go to Online Store → Themes
  2. Click "Edit Code" on your active theme
  3. Open theme.liquid
  4. Paste the Forgio script tag just before </head>
  5. Click Save
Custom HTML / Other
1 minute
  1. Open your site's main HTML file (usually index.html)
  2. Paste the Forgio script tag just before the closing </body> tag
  3. Save and deploy

Works with any static site, Next.js, Gatsby, Hugo, Jekyll, or any framework that outputs HTML.

Google Tag Manager
3 minutes
  1. Create a new Custom HTML tag
  2. Paste the Forgio script tag
  3. Set trigger to "All Pages"
  4. Click Save, then Submit and Publish
